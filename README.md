### 🧠 Features:

- Click to copy color to clipboard (`navigator.clipboard.writeText()`)

- 🌤️ Dynamic weather icons based on real-time conditions  
- ❌ Error handling for invalid city inputs  
- 💡 Responsive UI for different screen sizes
